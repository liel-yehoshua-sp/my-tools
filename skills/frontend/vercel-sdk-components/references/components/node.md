# Node

- **Category**: Workflow
- **Docs**: [https://elements.ai-sdk.dev/components/node](https://elements.ai-sdk.dev/components/node)
- **Install**: `npx ai-elements@latest add node`

## Description

Workflow node with input/output ports and custom content.

## Tags

`node, block, step, port, input-output`

## Scenarios

### ✅ Good Fit

- Building visual workflow/pipeline editors
- Creating agent orchestration UIs
- Node-based AI flow builders

### ❌ Bad Fit (with alternatives)

- Full-featured flow editors → Use React Flow or dagre directly
- Simple flowcharts → Use Mermaid diagrams
- Static diagrams → Use SVG or d3
