# Toolbar

- **Category**: Workflow
- **Docs**: [https://elements.ai-sdk.dev/components/toolbar](https://elements.ai-sdk.dev/components/toolbar)
- **Install**: `npx ai-elements@latest add toolbar`

## Description

Action toolbar for workflow operations like add node, delete, etc.

## Tags

`toolbar, actions, buttons, workflow-controls`

## Scenarios

### ✅ Good Fit

- Building visual workflow/pipeline editors
- Creating agent orchestration UIs
- Node-based AI flow builders

### ❌ Bad Fit (with alternatives)

- Full-featured flow editors → Use React Flow or dagre directly
- Simple flowcharts → Use Mermaid diagrams
- Static diagrams → Use SVG or d3
