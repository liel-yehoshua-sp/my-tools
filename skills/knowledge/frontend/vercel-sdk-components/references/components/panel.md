# Panel

- **Category**: Workflow
- **Docs**: [https://elements.ai-sdk.dev/components/panel](https://elements.ai-sdk.dev/components/panel)
- **Install**: `npx ai-elements@latest add panel`

## Description

Side panel overlay on the workflow canvas for configuration.

## Tags

`panel, sidebar, overlay, config, properties`

## Scenarios

### ✅ Good Fit

- Building visual workflow/pipeline editors
- Creating agent orchestration UIs
- Node-based AI flow builders

### ❌ Bad Fit (with alternatives)

- Full-featured flow editors → Use React Flow or dagre directly
- Simple flowcharts → Use Mermaid diagrams
- Static diagrams → Use SVG or d3
